# 977. 有续数组的平方

# 题目：给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。（有负数）

示例 1：

  • 输入：nums = [-4,-1,0,3,10]
  • 输出：[0,1,9,16,100]
  • 解释：平方后，数组变为 [16,1,0,9,100]，排序后，数组变为 [0,1,9,16,100]

示例 2：

  • 输入：nums = [-7,-3,2,3,11]
  • 输出：[4,9,9,49,121]

视频链接：https://www.bilibili.com/video/BV1QB4y1D7ep/?spm_id_from=333.337.search-card.all.click&vd_source=e0e7d9473b1926d5432c3c9db17fd187

文章链接：https://programmercarl.com/0977. 有序数组的平方.html# 算法公开课

我的第一想法：平方后再用 sort 排序。（时间复杂度太高了）

# 双指针解法

解决办法：由于数组有正有负，所以要从两边开始，定义两个指针从两头开始填如果负数的平方大于正数的平方，那就先填负数。反之。填完后将该指针向中间移动。

易错点：i<=j, 因为 i=j 的时候可能还有一个元素。

# 209. 长度最小的子数组

# 题目：给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的 连续 子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。

示例：

  • 输入：s = 7, nums = [2,3,1,2,4,3]
  • 输出：2
  • 解释：子数组 [4,3] 是该条件下的长度最小的子数组。

提示：

  • 1 <= target <= 10^9
  • 1 <= nums.length <= 10^5
  • 1 <= nums[i] <= 10^5

视频链接：https://www.bilibili.com/video/BV1tZ4y1q7XE/?spm_id_from=333.788&vd_source=e0e7d9473b1926d5432c3c9db17fd187

文章链接：https://programmercarl.com/0209. 长度最小的子数组.html# 思路

第一眼：把所有的情况遍历一遍。（但不知道怎么做）

看视频开头后：可以用两个 for 循环，一个代表左指针（i），一个是右指针（j），每到 j 移动时，就将它所对应的数字求和（sum），当和大于给定数字时记录 j-i 的长度。

解析：窗口就是满足最小长度的数组，> 缩 i，< 扩 j。

# 小工具：1.Math.min（a,b）可以比较出最小值。 2.Integer.MAX_VALUE 可以得出 int 类型最大的数，当判断是否执行数不能 - 1 时用这个标记。

易错点：1.lon 不能为 - 1，因为后面是比长度谁小。

2.long 是关键字（细节错误，下回用 result）

3. 不能用 if 因为 i 不确定要移动几次。

4. 求长度的时候要 + 1

# 题目：给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O (1) 额外空间并原地修改输入数组。

# 双指针法

双指针法（快慢指针法）： 通过一个快指针和慢指针在一个 for 循环下完成两个 for 循环的工作。

定义快慢指针

  • 快指针：寻找新数组的元素 ，新数组就是不含有目标元素的数组
  • 慢指针：指向更新 新数组下标的位置
